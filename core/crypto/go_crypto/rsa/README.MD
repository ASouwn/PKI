## GenerateKeyPair：

- 使用 rsa.GenerateKey 生成 RSA 密钥对。
- 将私钥和公钥编码为 PEM 格式。

## Encrypt：

- 使用 RSA 公钥加密数据，采用 OAEP 填充模式。

## Decrypt：

- 使用 RSA 私钥解密数据，采用 OAEP 填充模式。

## Sign：

- 使用 RSA 私钥对数据的哈希值进行签名。

## Verify：

- 使用 RSA 公钥验证签名。

`密钥长度`：示例中使用 2048 位密钥，可以根据需要调整（如 4096 位）。

`错误处理`：在实际应用中，请确保正确处理所有可能的错误。

`性能`：RSA 加密和解密操作较慢，适合加密小数据或用于密钥交换。
